手表 UX 流程

主页面：显示数字 + 添加按钮，水位动画直观展示进度
圆形屏幕：左/右滑动 → 进入菜单页，再滑动返回
方形屏幕：点击右上角 ••• → 弹出菜单，按钮变×关闭
添加水：点击 + 按钮 → 数字动画 + 水位平滑上升
减少水：长按 + 按钮 → 显示 - 选项

移动端竖屏 UX 流程

主页面：全屏水位 + 数字 + 添加按钮 + 底部Tab栏
Tab栏交互：点击切换页面，Tab栏收缩为返回按钮（300ms动画）
添加水：点击 + → 数字弹跳动画 + 水位平滑上升 + 涟漪效果
减少水：长按 + 按钮 → 显示 - 选项
历史页：显示今日记录列表，左滑删除单条记录
设置页：开关/选择项，点击返回按钮回到主页

移动端横屏 UX 流程

布局：左右各50%，左侧主卡片 + 右侧历史面板
主卡片：水位 + 数字 + 添加按钮
右侧面板：今日记录列表 + 底部Tab栏悬浮
Tab切换：点击Tab → 右侧面板内容切换，Tab栏变为返回按钮
优势：一屏同时看到进度和历史，无需切换页面

折叠屏/平板 UX 流程

布局：左右各50%，充分利用大屏空间
左侧：主卡片（水位 + 数字 + 添加按钮）
右侧：功能面板（历史/设置/统计）+ Tab栏
Tab切换：点击Tab → 右侧内容切换，动画过渡
二级页面：左侧卡片半透明，右侧面板内容替换，Tab栏变为返回按钮
优势：主界面始终可见，操作更高效

交互规范总结

交互 行为 动画
点击 + 按钮 添加一杯水 数字弹跳 + 水位上升 + 涟漪
长按 + 按钮 显示 - 选项 弹出菜单
点击 Tab 切换页面 Tab栏 → 返回按钮（300ms）
点击返回 回到主页面 返回按钮 → Tab栏（300ms）
点击菜单按钮 弹出菜单 按钮变为 × (自发光)
左滑记录 删除单条记录 滑出 + 淡出
手表滑动 进入/退出菜单 页面滑动切换
