# Water Reminder - 液态玻璃设计规范

## 设计理念

**核心隐喻：水位 = 进度**

用户喝水，水位上升。直观、自然、沉浸。

## 水效果系统

### 水位动画

```typescript
const progress = (today / goal) * 100
const waterHeight = windowHeight * progress / 100

// 动画
transition: {
  duration: 1.2,
  ease: [0.32, 0.72, 0, 1] // Apple ease
}
```

### 波浪

- **三层波浪**：前景、中景、背景
- **参数**：
  - 前景：振幅 6px, 频率 3, 速度 1.5x
  - 中景：振幅 5px, 频率 2.5, 速度 1.2x
  - 背景：振幅 4px, 频率 2, 速度 0.9x

### 气泡

- 随机生成：每 800ms 一个
- 大小：4-12px
- 上升时间：8-12秒
- 效果：径向渐变 + 内发光

### 光线

- **光束**：三道斜射光线，从水面透入
- **焦散**：椭圆渐变，模拟水底光斑
- **水面高光**：顶部白色渐变

### 涟漪

点击添加水时，水面产生扩散涟漪：
```typescript
// 涟漪动画
initial: { width: 0, height: 0, opacity: 1 }
animate: { width: 200, height: 200, opacity: 0 }
duration: 2s
```

## 交互设计

### 主操作

```
┌─────────────────────────┐
│         [−]  [+]  [⏱]  │
│              │          │
│         添加水          │
│        (涟漪效果)       │
└─────────────────────────┘
```

### 手势

- **点击 +**：添加一杯水，水位上升，产生涟漪
- **点击 −**：减少一杯水，水位下降
- **点击历史**：查看今日记录
- **点击设置**：打开设置面板

### 反馈

1. **视觉**：水位平滑上升
2. **涟漪**：水面扩散动画
3. **数字**：弹性缩放动画

## 设计决策

### 为什么用水位？

- **直观**：水越多 = 喝得越多
- **沉浸**：用户"在水中"
- **目标感**：水位到顶 = 目标达成

### 为什么用毛玻璃卡片？

- **不遮挡**：透明显示水位
- **现代**：Apple 液态玻璃语言
- **层次**：前景卡片 vs 背景水面

### 为什么三层波浪？

- **深度**：模拟真实水面
- **动感**：不同速度产生自然流动
- **美观**：渐变叠加更有质感
