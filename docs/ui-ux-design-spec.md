# 喝水提醒 UI/UX 设计规范

## 统一设计语言

### 核心样式
- **液态玻璃效果**: `background: linear-gradient(135deg, rgba(255,255,255,0.6) 0%, rgba(255,255,255,0.3) 100%); backdrop-filter: blur(20px);`
- **自发光按钮**: `box-shadow: 0 0 24px rgba(0,102,255,0.5)`
- **主色调**: `#0066ff`
- **水位渐变**: `rgba(77, 166, 255, 0.5) → rgba(0, 102, 204, 0.6) → rgba(0, 61, 122, 0.7)`
- **字体**: 梦源黑体 (Dream Source Han Sans)

### 背景色
- **浅色模式**: 白色半透明渐变 `linear-gradient(135deg, rgba(255,255,255,0.6) 0%, rgba(255,255,255,0.3) 100%)`
- **深色模式**: 深色背景 `#0d1b2a` 或 `#1b2838`
- **设置页面**: 纯白色 `white` 或纯深色

### 卡片样式
- **背景**: `linear-gradient(135deg, rgba(255,255,255,0.6) 0%, rgba(255,255,255,0.3) 100%)`
- **边框**: `border: 1px solid rgba(0,0,0,0.06)`
- **圆角**: `16px` (卡片), `32px` (大卡片)
- **阴影**: `box-shadow: 0 4px 20px rgba(0,0,0,0.1)`
- **模糊**: `backdrop-filter: blur(10px)` 或 `blur(20px)`

### 按钮样式
- **圆形按钮**: 
  - 背景: `linear-gradient(135deg, rgba(255,255,255,0.6) 0%, rgba(255,255,255,0.3) 100%)`
  - 边框: `border: 1px solid rgba(0,0,0,0.06)`
  - 激活状态: `box-shadow: 0 0 16px rgba(0,102,255,0.5)` (自发光效果)
- **添加按钮**: 
  - 直径: 64px (大), 56px (中), 48px (小)
  - 发光: `box-shadow: 0 0 24px rgba(0,102,255,0.5)`
  - 颜色: `#0066ff`

## 统一交互逻辑

### Tab栏
- 始终居中，左右按钮使用绝对定位，不影响Tab栏位置
- 尺寸: 120×40px (移动端)
- 样式: `rounded-2xl`, `backdrop-filter: blur(10px)`
- 激活项: `background: rgba(0,102,255,0.15)`

### 菜单按钮
- 点击弹出菜单后变为 × 关闭按钮
- 带自发光效果: `box-shadow: 0 0 16px rgba(0,102,255,0.5)`
- 直径: 44px (移动端), 40px (手表)

### 页面切换
- 全屏页面切换
- 返回按钮底部居中
- 直径: 44px

### 添加水
- 数字弹跳动画
- 水位平滑上升 (1.2s ease-out)

### 历史视图
- 日/周/月视图切换
- 周视图显示柱状图
- 月视图显示日历网格

### 设置页面
- 全屏页面，语言为二级菜单
- 背景: 纯白/深色 (不是玻璃效果)

### 分屏布局
- 宽度≥600px且高度≥500px时启用
- 元素保持固定尺寸居中显示，不随屏幕拉伸
- 留白处理更优雅

---

## 布局类型

### 1. 移动端竖屏 (width < 600px)
- 全屏水位动画
- 底部Tab栏 (主页/历史)
- 左侧视图切换按钮，右侧菜单按钮
- 添加按钮居中显示

### 2. 移动端横屏 (width >= 600px, height < 500px)
- 左右分屏各50%
- 左侧: 主卡片 (水位+添加) 始终可见
- 右侧: 历史面板
- Tab栏隐藏 (左侧卡片即主页)
- 左侧按钮: 视图切换
- 右侧按钮: 更多菜单(设置/关于)

### 3. 桌面分屏 (width >= 600px, height >= 500px)
- 左侧卡片固定尺寸 (320×384px)
- 右侧面板固定尺寸
- 元素居中，不随屏幕拉伸
- Tab栏隐藏

### 4. 手表 (圆形/方形屏幕)
- 全屏水位动画
- 菜单按钮右上角
- 全屏菜单覆盖层
- 返回按钮底部居中

---

## 视图类型

### 日视图
- 记录列表: 时间 + 杯数
- 左滑删除记录
- 日期导航: 左右箭头切换

### 周视图
- 柱状图显示每日完成量
- 当天高亮显示
- 统计: 总计/日均/达成率

### 月视图
- 日历网格显示
- 颜色表示完成度:
  - 达标: `rgba(0,102,255,0.8)` 或 `#0066ff`
  - 部分: `rgba(0,102,255,0.4-0.6)`
  - 未完成: 无背景
- 当天边框高亮: `border: 2px solid #0066ff`

---

## 菜单结构

### 手表端
```
主界面 → 历史 → 日/周/月视图
       → 设置 → 深色模式开关
```

### 移动端/桌面端
```
主界面 → Tab栏 → 主页/历史
       → 视图按钮 → 日/周/月 (弹出菜单)
       → 菜单按钮 → 设置/关于 (弹出菜单)

设置页面 → 深色模式开关
        → 语言 → 中文/English (二级页面)
        → 每日目标 → 调整目标 (二级页面)
        → 关于 → 应用信息
```

## 颜色规范

### 浅色模式
- 背景: `linear-gradient(135deg, rgba(255,255,255,0.6) 0%, rgba(255,255,255,0.3) 100%)`
- 文字: `#1a1a1a` (主要), `#666` (次要), `#999` (辅助)
- 水位: `rgba(77, 166, 255, 0.5) → rgba(0, 102, 204, 0.6) → rgba(0, 61, 122, 0.7)`
- 强调色: `#0066ff`
- 边框: `rgba(0,0,0,0.06)`

### 深色模式
- 背景: `#0d1b2a` 或 `#1b2838`
- 文字: `white` (主要), `rgba(255,255,255,0.8)` (次要), `rgba(255,255,255,0.6)` (辅助)
- 水位: 同浅色模式
- 强调色: `#4fc3f7`
- 边框: `rgba(255,255,255,0.1)` 或 `rgba(255,255,255,0.2)`

## 动画规范

### 水位上升
- 持续时间: 1.2s
- 缓动函数: ease-out
- 波浪动画: 持续循环

### 数字弹跳
- 初始: scale 1.1, opacity 0
- 结束: scale 1, opacity 1
- 持续时间: 0.2s

### 菜单弹出
- 初始: opacity 0, y 10
- 结束: opacity 1, y 0
- 持续时间: 0.2s
- 类型: spring, damping 25, stiffness 300

### 按钮点击
- scale: 0.9
- 持续时间: 短暂
